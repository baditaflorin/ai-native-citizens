# Slide Template Notes

Slides are not the project. Use them only to support the live demo.

## 5-Minute Demo Structure

### Slide 1: Project Name

Project name, team, challenge, city.

### Slide 2: User And Problem

Name the user specifically. State the problem in one sentence.

### Slide 3: Prototype

One sentence on what it does. Then switch to the live demo.

### Slide 4: What Works / What Does Not

Be direct. Judges should not have to discover the limits by accident.

### Slide 5: Build Log And Next Step

Show build-log link, repo/demo link, and the next 30-day ask.

## Visual Direction

- Cream background.
- Near-black text.
- One vermillion accent.
- Fraunces headline.
- Public Sans body.
- JetBrains Mono for links or build-log label.

## Do Not Include

- Fake traction.
- Generic AI art.
- Vendor logos unless tools are actually used.
- Long biographies.
- More than one chart unless the prototype is data-first.
- Claims without sources.
