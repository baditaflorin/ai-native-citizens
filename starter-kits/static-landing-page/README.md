# Static Landing Page Starter

No build step. No analytics. No third-party scripts. This starter gives sprint teams a public project page with problem, user, demo, source, limits, build log, and team credits.

## Use

Edit `project.config.json`, then open `index.html` in a browser.

## Publish

Host on GitHub Pages, Netlify, a static bucket, or any ordinary web server. Keep the build-log link visible.
