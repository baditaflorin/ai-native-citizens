# ADR-0002: Use CC-BY-4.0 for Docs, MIT for Code, and Trademark Control for Brand

## Context

AI-Native Citizens needs open replication. Local organizers should be able to copy, translate, remix, and run the kit without permission. At the same time, official sprint status, certification, and partner claims need protection so the public can tell community editions from approved chapters.

## Decision

Documentation, templates, translations, and ADRs are licensed under Creative Commons Attribution 4.0. Code is licensed under MIT. The AI-Native Citizens name and related marks remain trademark-controlled, with enforcement rules kept light in v0.1 and clarified as the movement grows.

## Consequences

The kit can spread fast while keeping a boundary around official identity. Contributors can reuse the material in cities, schools, NGOs, and universities. Code snippets and starter kits remain easy to fork. Organizers must not imply official approval unless approved by the steward team.

## Alternatives Considered

CC0 was too permissive for attribution and brand context. GPL-style code licensing was too heavy for starter kits. Fully closed brand and materials would block the movement before it had evidence from real events.
