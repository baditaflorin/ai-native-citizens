# ADR-0004: Use Punk-Civic Typography, Newsprint Cream, Near-Black Ink, and Vermillion

## Context

The visual system must separate AI-Native Citizens from generic AI products, SaaS pages, and conference branding. The project needs to feel civic, public, legible, and a little confrontational without becoming chaotic.

## Decision

The aesthetic direction is typography-first: Fraunces for display, Public Sans for body, JetBrains Mono for build-log and ADR accents. The locked palette is warm newsprint cream `#F5F2EC`, near-black `#0A0A0A`, electric vermillion `#FF3D00`, warm grey `#6B665E`, and rule `#2A2826`.

## Consequences

The site and materials read like a civic manifesto poster rather than a software startup. The accent color must be used sparingly so it keeps force. Layouts should use space, rules, and type scale instead of illustrations, gradients, stock photos, or decorative UI effects.

## Alternatives Considered

Generic AI gradients were rejected as category noise. Corporate clean layouts were rejected because they soften the politics of the work. Illustration-heavy systems were rejected because the project needs public artifacts, not decorative explanation.
