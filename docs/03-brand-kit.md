# Brand Kit

AI-Native Citizens is a civic manifesto brand, not a software vendor brand. It should feel public, sharp, typographic, and useful.

## Naming Hierarchy

| Layer | Name |
| --- | --- |
| Movement / identity | AI-Native Citizens |
| Short form | AI Natives |
| Local chapter | AI Natives of [City] |
| Event | AI-Native Citizens: [City] ([dates]) |
| Kit | The AI-Native Citizens Kit |
| Method | The Civic AI Sprint |
| Annual global gathering | AI Natives Summit |
| Tagline | none |
| Action verb | Become an AI-Native Citizen |

Use "AI-Native Citizens" with the hyphen on first reference. "AI Natives" is acceptable in casual context after the full name is established. Do not write "AI Citizens" in user-facing copy.

## Color Palette

```css
--bg:     #F5F2EC; /* warm newsprint cream */
--ink:    #0A0A0A; /* near-black */
--accent: #FF3D00; /* electric vermillion */
--muted:  #6B665E; /* warm grey for meta */
--rule:   #2A2826; /* hairline borders */
```

Use the accent sparingly: one primary CTA, blockquote markers, small headline accents, and colophon mark. If the page starts to glow vermillion, it is wrong.

## Typography

Display: Fraunces, weights 300, 600, 900, optical size 144 for hero.

Body: Public Sans, weights 400, 500, 700.

Mono: JetBrains Mono, weight 400, for ADR labels, build logs, version text, and code accents.

Never substitute Inter, Roboto, Space Grotesk, or system UI for the public site. The type choices are part of the civic signal.

## Wordmark Usage

The wordmark is typographic: "AI-Native Citizens" in Fraunces. Keep the hyphen. Keep the full name visible on first view for public materials.

Do not compress it into initials except for the favicon or tiny utility contexts. The favicon may use "AC" as a typographic mark.

## Social Asset Guidance

Use typographic posters: cream background, near-black ink, one vermillion accent. Use large words, strong rules, and direct event facts. Do not use stock photos, generic robot art, AI-glow gradients, or corporate conference templates.

## What Not To Do

- No generic AI gradient hero.
- No purple-on-white SaaS page.
- No stock photos of people at laptops.
- No AI-generated illustrations.
- No corporate clean polish.
- No heart-symbol footer line.
- No cookie banner.
- No "AI Citizens" shorthand.
- No slogan/tagline invented for convenience.
